# Purchase Request App

Este projeto faz parte do meu processo de aprendizagem de **SAP CAP (Cloud Application Programming Model)** usando o **SAP Build Code** e o **Business Application Studio (BAS)**.

A ideia inicial é começar com conceitos muito básicos do CAP e evoluir gradualmente até criar uma aplicação fullstack completa.

## Objetivo

O objetivo deste projeto é criar uma aplicação simples de **Purchase Requests**, onde será possível criar pedidos de compra e, numa fase posterior, implementar um fluxo de aprovação.

A aplicação será construída passo a passo, começando pelo backend com CAP e depois avançando para o frontend.

## O que estou a aprender

Neste projeto estou a explorar conceitos como:

- Criação de entidades em CAP
- Serviços OData
- Eventos como `CREATE`, `READ`, `UPDATE` e `DELETE`
- Validações no backend
- Lógica de negócio com handlers em JavaScript
- Integração futura com frontend
- Construção de uma aplicação fullstack com SAP CAP

## Funcionalidades planeadas

- Criar pedidos de compra
- Validar valores introduzidos
- Listar pedidos existentes
- Atualizar pedidos
- Apagar pedidos
- Adicionar estado ao pedido, por exemplo: `Pending`, `Approved`, `Rejected`
- Implementar aprovação de pedidos
- Criar uma interface frontend para interagir com a aplicação
